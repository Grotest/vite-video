## Setup

Make sure to install the dependencies:

```bash
# 
git clone git@github.com:Grotest/vite-video.git

# npm
npm install

# npm
npm install -g vite

# npm if you do not have the current version node
nvm use node 
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
